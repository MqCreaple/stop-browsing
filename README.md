# 别刷了

Edge / Chrome 浏览器扩展，监控多平台网站使用时长，超限后强制暂停。

## 支持的网站

| 平台 | URL 匹配 |
|------|----------|
| ✅ YouTube  | `youtube.com/watch?v=` |
| ✅ B站      | `bilibili.com/video/` |
| ✅ TikTok   | `tiktok.com/@*/video/` |
| ✅ 抖音     | `douyin.com/video/` |
| ✅ 小红书   | `xiaohongshu.com`（全部页面） |
| ✅ 知乎     | `zhihu.com`（全部页面） |
| ✅ 微博     | `weibo.com`（全部页面） |
| ✅ 百度贴吧 | `tieba.baidu.com`（全部页面） |
| ✅ Twitter/X | `x.com`（全部页面） |

每个站点可在设置中独立开关，默认全部开启。

## 功能

- ⏱ **自动计时** — 检测标签页是否为视频/内容页面，自动累计停留时间（跨站点合计）
- 🔄 **跨会话持久化** — 关闭标签页后时间不丢失，每日零点自动重置
- 🚨 **强制暂停** — 超限后弹出全屏覆盖层，拦截所有输入，需完成任务后方可关闭
- 🌙 **夜间加速** — 23:00–06:00 自动切换为更短限制（默认 20 分钟，可自定义）
- 🧩 **暂停行为** — 可选多种暂停模式：倒计时等待、四则运算答题、Tripos 数学真题
- ⚙️ **完全可配置** — 站点开关、夜间时段、日间/夜间时长、暂停时长、冷却间隔均可调
- 📊 **历史记录** — 弹窗中查看最近 7 天每日总使用量

## 暂停行为

| 行为 | 说明 |
|------|------|
| ⏱ 倒计时 | 等待指定时间后自动恢复（默认） |
| 🧮 四则运算 | 完成 10 道算术题后才能恢复 |
| 📐 Tripos 数学真题 | 查看随机历年剑桥数学真题，倒计时后确认完成 |

## 安装

### Edge

```
edge://extensions → 开启「开发人员模式」→ 加载解压缩的扩展 → 选择 stop-browsing/
```

### Chrome

```
chrome://extensions → 开启「开发者模式」→ 加载已解压的扩展程序 → 选择 stop-browsing/
```

## 使用

1. 安装后点击工具栏 🦞 图标打开监控面板
2. **监控页** — 查看今日累计使用量、各站点明细、最近 7 天记录
3. **设置页** — 开关站点、选择暂停行为、配置夜间时段、调整时间限制
4. 达到限制时，正在浏览的页面自动弹出强制暂停覆盖层
